# Club Center Fight Academy - Sitio Web

## 🥋 Descripción
Sitio web profesional para Club Center Fight Academy, una academia de artes marciales ubicada en Buenos Aires, Argentina. El sitio incluye información sobre disciplinas, horarios, contacto y un sistema híbrido de atención al cliente.

## ✨ Características Principales

### 🎨 Diseño Responsivo Optimizado
- **Vista móvil completamente rediseñada** con navegación moderna
- **Header sticky** con menú hamburguesa para móviles
- **Barra de WhatsApp flotante** en dispositivos móviles
- **Grid system moderno** con CSS Grid y Flexbox
- **Animaciones suaves** y transiciones optimizadas

### 📱 Optimizaciones Móviles
- **Navegación móvil mejorada** con menú deslizante
- **Botones táctiles optimizados** (44px mínimo)
- **Tipografía escalable** y legible en todas las pantallas
- **Imágenes responsivas** con lazy loading
- **Performance optimizada** para dispositivos móviles

### 💬 Sistema de Atención Automatizado
- **Asistente virtual** para consultas generales y automatización
- **Contacto directo** solo para formulario de contacto específico
- **Botones flotantes** estratégicamente ubicados
- **Integración con Google Analytics** para tracking

### 🎯 Funcionalidades
- **Páginas principales**: Inicio, Disciplinas, Nosotros, Contacto
- **Formulario de contacto** integrado con WhatsApp
- **Galería de testimonios** con carrusel interactivo
- **Mapa de ubicación** integrado
- **Horarios de clases** con imagen ampliable
- **Disciplinas disponibles**: Jiu-Jitsu, Muay Thai, Boxeo, MMA

## 🛠️ Tecnologías Utilizadas

### Frontend
- **HTML5** semántico y accesible
- **CSS3** con variables CSS y Grid/Flexbox
- **JavaScript ES6+** modular y optimizado
- **Font Awesome** para iconografía
- **Google Fonts** (Montserrat, Oswald)
- **Git** para control de versiones
- **Google Analytics** para métricas
